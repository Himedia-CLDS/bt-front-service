### 라이브러리
```
npm install react-router-dom
```
### 시작
```
npm start
```

### .env작성
```
REACT_APP_IMG_BASE_URL = ...
REACT_APP_API_BASE_URL = ...
```

### URL
| URL                                              | 설명             |
|--------------------------------------------------------|----------------|
| /                     | 메인 모든상품정보 조회   |
| /product               | 상품 상세조회   |
| /loginsingup/:type               | login: 로그인/ signup:회원가입   |
| /mypage               | 내정보   |
| /feed               | 피드조회   |
| /create               | 피드작성   |
